NLP Pipeline (Sajjad)

Trigger Data Ingestion & Batch Embedding

Initiates the offline pipeline to scrape Akoma Ntoso XML files, clean the text, and execute the Legal-BERT batch embedding to generate 1x768 vectors for the FAISS index. [Achieves Requirements OP 1.1, OP 1.2]

Authorizations:
InternalApiKeyAuth
Request Body schema: application/json
required
target_acts
required
Array of strings

List of exact Akoma Ntoso FRBR URIs or standard act names to scrape from new.kenyalaw.org.

force_refresh
boolean
Default: false

If true, bypasses repeal quarantines and overwrites existing raw text in PostgreSQL.

Topic Modeling (Horris)

Trigger BERTopic Clustering

Executes UMAP dimensionality reduction (768d -> 5d) and HDBSCAN density clustering on the pre-computed vectors. Updates the latent_theme_id column in the PostgreSQL relational database. [Achieves Requirement F.2.1]

Authorizations:
InternalApiKeyAuth
Request Body schema: application/json
required
embedding_version
required
string
min_cluster_size
integer
Default: 15

Dynamic calibration parameter for the HDBSCAN algorithm.
